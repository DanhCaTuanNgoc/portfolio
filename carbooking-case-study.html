Description

This project was developed in collaboration with a real client, Nhut Truong Interprovincial Car Service, a transportation business providing long-distance vehicle rental services across provinces in Vietnam.

The system is designed as a lightweight O2O platform where customers can search for routes, view pricing, and initiate booking via Zalo instead of completing transactions within the system.

The primary goal is to improve business visibility, provide transparent pricing, and streamline customer interaction while keeping operational complexity low.

Role - Business Analyst (BA)

  • Conducted requirement elicitation with stakeholders to define workflows and project scope.
  • Authored SRS with 5 Epics, 5 User Stories, and 20+ Acceptance Criteria.
  • Modeled business workflows using BPMN for customer search and O2O booking flow.
  • Designed ERD for route pricing and vehicle classification.
  • Defined business rules such as duplicate route prevention and pricing logic.
  • Proposed technical solutions including in-memory filtering and secure authentication with bcrypt.

Business Capabilities

The solution is structured around practical capabilities that directly support O2O operations.

Search and Filtering

  • Search routes by province or destination with case-insensitive matching
  • Filter results by configurable price ranges
  • Instant UI updates using in-memory filtering without API reload
  • Supports empty states and no-result scenarios

Pricing Visualization

  • Displays one-way and round-trip prices clearly
  • Supports route-level price comparison for quick decisions
  • Promotes transparent pricing for customer trust

Zalo Booking (O2O)

  • Redirects users to Zalo for booking interaction
  • No internal booking or payment implementation
  • Maintains a lightweight and practical O2O model

Admin Data Management

  • CRUD for car types, routes, posts, and featured locations
  • Bulk import route data via Excel templates
  • Duplicate route prevention by composite key: province + destination + car type

Authentication and Security

  • Admin login with secure password hashing using bcrypt
  • Change password workflow for account control
  • API-based authentication flow for protected management actions
