Description

AutoEdu is a modern learning management system designed for educational institutions to manage courses, lectures, assignments, examinations, and student performance in one unified platform.

The system follows a mobile-first approach and supports real-time updates, enabling students, teachers, and administrators to collaborate effectively with consistent data synchronization.

The primary goal is to digitize class operations end-to-end while ensuring usability, scalability, and secure role-based access across all workflows.

Role - Business Analyst (BA)

  • Defined product scope and role capabilities for Student, Teacher, and Admin actors.
  • Authored Software Requirements Specification (SRS) covering functional requirements, non-functional constraints, and acceptance criteria.
  • Produced key system design artifacts including Entity-Relationship Diagram (ERD), BPMN, and 3 Use Case Diagrams.
  • Specified API boundaries for authentication, class, assignment, and file workflows.
  • Reached 100+ registered users within the department, validating core learning and assessment workflows.
  • Aligned product artifacts with implementation stack: Next.js, TypeScript, Supabase, Tailwind CSS.

Business Capabilities

The solution is structured around practical modules for digital learning operations.

Student Learning Workspace

  • Access enrolled classes, lecture materials, and progress information
  • Submit assignments with file attachments and deadline status tracking
  • Join online exams with timed attempts and multiple question types
  • Receive real-time notifications for upcoming tasks and announcements

Teacher Teaching Hub

  • Create and manage classes, enrollment, and teaching materials
  • Publish lectures using rich text, documents, media links, and resources
  • Design assignments and exams with configurable rules and due windows
  • Grade submissions with structured feedback and performance insights

Administration and Governance

  • Manage user lifecycle for students, teachers, and administrators
  • Configure permissions and monitor platform activity
  • Generate reports for class outcomes and institutional analytics

Data and API Architecture

  • PostgreSQL schema for profiles, classes, enrollments, lectures, assignments, and exams
  • RESTful endpoints for authentication, class CRUD, submissions, and file management
  • Supabase storage integration for upload and retrieval workflows

Security and Compliance

  • JWT-based authentication and role-based access control
  • Row-level security policies to enforce scoped data visibility
  • Input validation and common web security controls for robust operations

Performance and UX

  • Next.js App Router with code splitting and lazy-loading strategies
  • Responsive interface for mobile and desktop learning scenarios
  • Accessible component design aligned with modern UX standards
